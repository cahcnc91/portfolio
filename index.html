Puppydate

This a fully responsive app that mimics Tinder usage, but for puppies. Built using Nodejs, React, Redux and MongoDB.Just like Tinder, the user can go through profiles and choose to match or not with another puppy. In case of a match from both, a chat is created and you guys can talk to set up playdates for the puppies. Real-time chatting using Socket.io.
